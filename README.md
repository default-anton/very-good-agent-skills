# very-good-agent-skills

These skills are mine. I like them, and maybe you will too.

This repo is a small pile of reusable agent skills I keep around because they make the work better. They are short and practical.

## Included

- `agents-md`: rules for creating and editing `AGENTS.md`
- `create-cli`: design command-line interface parameters and UX
- `git-commit`: workflow and constraints for creating git commits

## Installation

For pi users:

```bash
pi install git:github.com/default-anton/very-good-agent-skills
```

Otherwise, copy the skill directory you want into your tool's skills directory:

```bash
curl -L https://codeload.github.com/default-anton/very-good-agent-skills/tar.gz/refs/heads/main \
  | tar -xz
cp -R very-good-agent-skills-main/agents-md ~/.codex/skills/
```
